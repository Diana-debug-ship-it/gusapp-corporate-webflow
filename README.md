# GusApp International — Корпоративный B2B-сайт IT-компании

Коммерческий проект корпоративного сайта, разработанный на платформе Webflow для международной IT- и маркетинговой компании **GusApp International FZCO** (Дубай, ОАЭ). Сайт презентует услуги компании в сфере разработки CRM-систем, SaaS-решений, кастомного ПО и маркетинговых стратегий для глобального рынка.

* **Статус проекта:** В продакшене, является официальной визитной карточкой компании.
* **Ссылка на живой сайт:** [gusappint.com](https://gusappint.com/)
* **Инструменты разработки:** Webflow CMS, JavaScript, кастомная верстка элементов интерфейса, интеграция внешних форм.

---

## 💼 Какую бизнес-задачу решает проект
В отличие от быстрых потребительских лендингов, этот сайт работает в строгом B2B-сегменте с длинным циклом сделки:
1. **Позиционирование бренда:** Презентация экспертизы международной команды, ключевых метрик (50+ успешных проектов) и спектра сложных технологических услуг (SaaS, CRM, IT-консалтинг).
2. **B2B-лидогенерация:** Проектирование и настройка форм захвата «Let's Build Your Success Together» для сбора комплексных заявок на разработку программного обеспечения.
3. **SEO и международная доступность:** Чистая семантическая верстка на Webflow, оптимизированная под поисковые роботы (SEO-friendly) и быструю загрузку из любой точки мира.

## 🛠 Техническая реализация и кастомные решения
* **Интеграция корпоративной почты и уведомлений:** Данные из форм отправки заявок агрегируются и мгновенно перенаправляются на корпоративные адреса компании (`info@gusappint.com`) и внутреннюю CRM-систему для фиксации лидов.
* **Кастомный JavaScript:** Дописаны скрипты для плавной работы интерактивных блоков интерфейса (аккордеоны в разделе FAQ, кастомные анимации появления элементов).
* **Стилизация и адаптивность:** Прописаны кастомные CSS-стили для точной адаптации интерфейса под различные разрешения экранов (от широкоформатных мониторов до мобильных устройств).
